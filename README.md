# Library Database Analysis

## Project Overview
This project is a SQL-based Library Management and Analysis system designed to understand real-world library operations.

## Key Features
- Database design using ER diagram
- Tables for books, authors, publishers, borrowers, branches, and loans
- SQL queries for analysis and insights

## Tools Used
- SQL
- MySQL
- MySQL Workbench

## Key Insights
- Branch-wise book demand analysis
- Popular books and authors
- Borrower activity patterns
- Inventory distribution across branches
